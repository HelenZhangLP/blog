NodeJs 学习笔记

女魔头
2019-05-31
Javascript, node

path

提供用于处理文件路径和目录路径的实用工具

  • path.resolve()

    将路径或片断序列解析为绝对路径

  • path.join()

    使用平台特定分隔符将所有给定 path 片段连接在一起，规范化生成路径

    process

    全局变量，提供有关当前 Node.js 进程信息，并对其进行控制，不需要 require

    process - 信号事件

    Interrupt from keyboard
    SIGINT 在终端运行时，可以被所有平台支持，通常可以通过 +C 触发(虽然这个不能配置)。 当终端运行在raw模式，它不会被触发。
