CSS display: none vs visibility: hidden

2019-05-13

display:none 与 visibility:hidden 区别

display 指定元素显示框架的类型，none 隐藏不占文档流，不会影响布局
visibility 显示或隐藏元素而不更改文档的布局，hidden 相当于此元素变成透明并且 text-indent: -9999px，会影响布局
