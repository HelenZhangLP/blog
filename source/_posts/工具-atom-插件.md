## 使用

#### markdown-preview-plus
1. command + shift + p
2. markdown preview toggle

#### platformio-ide-terminal
1. command + shift + p
2. terminal new / terminal close

#### [docblockr](https://atom.io/packages/docblockr)
用于自动生成注释（annotation）

#### authcompletePaths
自动补全 path
